```
@startuml
title 选课
actor users as U
loop 教师或学生将多次循环选课
U -> choice_courses: 用户点击选课
alt 如果users是教师
    choice_courses --> U: 返回已发布课程的数据
    U -> choice_courses:点击选择按钮
    choice_courses --> U: 录入选课教师的工号、姓名和选课时间
else 如果users是学生
    choice_courses --> U: 返回已发布课程的数据
    U -> choice_courses: 根据查找课程id进行选课
    choice_courses --> U: 返回选课教师的信息等数据
    U -> choice_courses: 录入选课学生的学号、姓名和选课时间
end
choice_courses --> U: 返回选课成功
end


@enduml
```
